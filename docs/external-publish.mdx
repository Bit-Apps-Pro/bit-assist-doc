## Overview
External publish Using this feature, you can use the Bit Assist widget on another domain or subdomain. You can use your widget on other sites without using the Bit Assist plugin.

## Set up external Publish
**âž¦ Follow the below steps**

1.	First, create a widget; in the row of your created widget, you will see 'Use widget in' option. Select the external site there. 
  <Image src="external-publish/create-widget-and-select-use-widget-in" alt="Bit Assist" w="1046" h="200"/>

2.	Edit the widget and go to the external publish tab
  <Image src="external-publish/external-tab" alt="Bit Assist" w="1046" h="200"/>
 
3.	Copy the link of the domain where you want to use this widget. Paste in the add domain field, then click the right button
  <Image src="external-publish/add-domain" alt="Bit Assist" w="1046" h="200"/>
 
4.	Copy the script and Paste it just above the `<body />` tag on your website.
  <Image src="external-publish/copy-the-script" alt="Bit Assist" w="1046" h="200"/>
 
5.	Done. Now your Bit Assist widget live on another website.
